# Scrape Data Directory

This directory contains Truman State University degree requirement data used by the AI agent.

## Files

- `parsedTrumanReq.json` - Parsed Truman degree requirements in structured format
- `truman_degree.json` - Truman degree program information
- `truman_REQ.json` - Raw Truman requirement data
- `truman_events.json` - Academic calendar events
- `truReq-Schema.json` - JSON schema for requirement validation

These files are core to the application and are loaded by the AI agent to provide accurate academic planning advice.
