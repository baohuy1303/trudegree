# Examples Directory

This directory contains example outputs and logs from the AI agent for reference and testing purposes.

## Subdirectories

### `conversation-logs/`
Contains saved AI conversation logs showing different prompts and responses. These are useful for:
- Testing prompt engineering
- Comparing model performance
- Debugging AI responses

### `parsed-degrees/`
Contains example outputs from the DegreeWorks PDF parser showing various parsing scenarios and edge cases.

**Note:** These files are for development reference and are typically not needed in production.
