# Data Directory

This directory contains sample data and test files for the TruDegree backend.

## Subdirectories

### `sample-plans/`
Sample academic plans and course planning configurations used for testing and development.

### `test-pdfs/`
Test DegreeWorks PDF files used for development and testing the PDF parser.

**Note:** User-uploaded PDFs are stored in `backend/pdfs/` at runtime and are not committed to git.
